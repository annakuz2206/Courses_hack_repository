# Инструкция: Выбор роли и цвета команды

## Как это работает

При создании команды теперь появляется модальное окно для выбора роли с градиентным фоном.

### Шаги:

1. **Создание команды**
   - Перейдите на страницу создания команды
   - Введите название команды
   - Нажмите "Создать команду"

2. **Модальное окно выбора роли**
   - Появится модальное окно с градиентом от черного до фиолетового
   - Выберите роль из выпадающего списка (Frontend, Backend, Designer, ML Engineer, Fullstack)
   - Используйте слайдер "Уровень уверенности" для настройки интенсивности фиолетового цвета
   - Градиент меняется в реальном времени при движении слайдера

3. **Сохранение**
   - Нажмите "Подтвердить"
   - Выбранный цвет сохраняется вместе с командой

4. **Страница "Моя команда"**
   - Фон страницы автоматически использует выбранный градиент
   - Градиент сохраняется даже после перезагрузки страницы

## Технические детали

### Градиент
- **Начальный цвет**: Черный (#000000)
- **Конечный цвет**: Вычисляется на основе уровня уверенности
- **Формула**: `rgb(139 * confidence, 92 * confidence, 246 * confidence)`
- **Диапазон уверенности**: 0.0 - 1.0

### Примеры цветов:
- **0.0**: Полностью черный
- **0.5**: Темно-фиолетовый `rgb(70, 46, 123)`
- **0.8**: Средне-фиолетовый `rgb(111, 74, 197)`
- **1.0**: Яркий фиолетовый `rgb(139, 92, 246)`

## Что сохраняется

В объекте команды теперь хранятся:
- `name` - название команды
- `role` - выбранная роль
- `confidence` - уровень уверенности (0-1)
- `backgroundColor` - вычисленный цвет в формате `rgb(r, g, b)`

## Стилизация

Модальное окно включает:
- ✅ Плавную анимацию появления
- ✅ Градиентный фон, меняющийся в реальном времени
- ✅ Стилизованный выпадающий список
- ✅ Кастомный слайдер с фиолетовым ползунком
- ✅ Полупрозрачные элементы с эффектом размытия (backdrop-filter)
