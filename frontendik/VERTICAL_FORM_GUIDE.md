# Вертикальный компонент формы анкеты

## Описание

Создан вертикальный компонент формы заполнения анкеты для мобильного интерфейса в узком контейнере с высокой степенью скругления углов.

## Характеристики контейнера

### Размеры и форма
- **Ширина**: 280px (узкий вертикальный контейнер)
- **Высота**: 750px
- **Скругление углов**: 40px (высокая степень скругления)
- **Позиция**: Центрирован внутри экрана iPhone

### Визуальный стиль
- **Фон**: Полупрозрачный glassmorphism `rgba(255, 255, 255, 0.08)`
- **Размытие**: `backdrop-filter: blur(20px)`
- **Граница**: Тонкая светлая `rgba(255, 255, 255, 0.15)`
- **Тень**: Глубокая тень для объемности

## Элементы формы

### 1. Заголовок
- **Размер**: 26px
- **Цвет**: Фиолетовый `#AE00FF`
- **Эффект**: Текстовое свечение
- **Расположение**: Две строки, центрировано

### 2. Лейблы полей
- **Размер**: 14px
- **Стиль**: Жирный (600), uppercase
- **Цвет**: Белый
- **Межбуквенный интервал**: 0.5px

### 3. Поля ввода
- **Фон**: Темный полупрозрачный `rgba(0, 0, 0, 0.3)`
- **Граница**: Фиолетовая `rgba(174, 0, 255, 0.3)`
- **Скругление**: 20px
- **Padding**: 12px 16px
- **Размер текста**: 14px

**При фокусе:**
- Фон меняется на `rgba(174, 0, 255, 0.1)`
- Граница становится ярче `rgba(174, 0, 255, 0.6)`
- Появляется фиолетовое свечение

### 4. Навыки (теги)
- **Фон**: Полупрозрачный фиолетовый
- **Граница**: Фиолетовая
- **Скругление**: 12px
- **Размер**: 14px

### 5. Кнопка отправки
- **Фон**: Градиент `#AE00FF` → `#C026D3`
- **Скругление**: 25px (pill-образная)
- **Высота**: 48px
- **Текст**: Uppercase, жирный, с межбуквенным интервалом
- **Тень**: Фиолетовое свечение

## Структура CSS

```css
.form-container-q {
    width: 280px;
    height: 750px;
    background: rgba(255, 255, 255, 0.08);
    backdrop-filter: blur(20px);
    border: 1px solid rgba(255, 255, 255, 0.15);
    border-radius: 40px;
    overflow: hidden;
    box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4);
}
```

## Адаптация для мобильного

### Компактность
- Уменьшены отступы между полями (18px)
- Уменьшены размеры шрифтов
- Оптимизирована высота полей

### Прокрутка
- Вертикальная прокрутка внутри контейнера
- Скрытая полоса прокрутки
- Плавная прокрутка

### Визуальная иерархия
- Четкое разделение полей
- Контрастные лейблы
- Яркие акценты на интерактивных элементах

## Цветовая схема

| Элемент | Цвет | Использование |
|---------|------|---------------|
| Заголовок | #AE00FF | Основной акцент |
| Лейблы | #FFFFFF | Текст |
| Поля (фон) | rgba(0,0,0,0.3) | Фон полей |
| Поля (граница) | rgba(174,0,255,0.3) | Граница |
| Фокус | rgba(174,0,255,0.6) | Активное состояние |
| Кнопка | #AE00FF → #C026D3 | Градиент |

### Glassmorphism
- Полупрозрачный фон

### Свечение
- Заголовок: текстовое свечение
- Поля при фокусе: box-shadow свечение
- Кнопка: фиолетовое свечение тени

### Анимации

## Использование

Форма автоматически отображается в узком вертикальном контейнере внутри экрана iPhone 13 Mini. Все элементы оптимизированы для мобильного взаимодействия с удобной прокруткой и четкими визуальными акцентами.
