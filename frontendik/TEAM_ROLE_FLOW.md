# Инструкция: Выбор роли и уровня уверенности

## Как это работает

При создании команды теперь есть двухэтапный процесс выбора роли и уровня уверенности.

### Этап 1: Выбор роли

После ввода названия команды появляется модальное окно со списком ролей:
- Frontend
- Backend
- Designer
- ML Engineer
- Fullstack
- Product-менеджер

Выберите свою роль, кликнув на нее.

### Этап 2: Уровень уверенности

После выбора роли появляется второе модальное окно с:

1. **Заголовок**: "Насколько ты уверен в своих способностях?" (фиолетовый градиентный текст)

2. **Выбранная роль**: Отображается с галочкой в фиолетовом квадрате

3. **Градиентная полоса**: Интерактивный слайдер
   - Перемещайте мышкой по полосе
   - Градиент меняется от черного до фиолетового
   - Чем правее, тем ярче фиолетовый цвет

4. **Кнопка "Подтвердить"**: Большая фиолетовая кнопка для сохранения

### Результат

После подтверждения:
- Команда создается с выбранной ролью
- Цвет градиента сохраняется
- На странице "Моя команда" фон будет использовать выбранный градиент

## Дизайн

### Модальное окно уверенности:
- **Фон**: Черный (#000000)
- **Заголовок**: Градиент от светло-фиолетового (#A78BFA) до фиолетового (#8B5CF6)
- **Чекбокс**: Фиолетовый квадрат (#8B5CF6) с белой галочкой
- **Роль**: Белый текст, 32px
- **Градиентная полоса**: 
  - Высота: 50px
  - Граница: 2px фиолетовая с прозрачностью
  - Градиент: От черного до выбранного оттенка фиолетового
- **Кнопка**: Фиолетовый градиент, закругленная (50px radius)

### Анимация:
- Модальное окно появляется снизу с плавным увеличением
- Градиент меняется в реальном времени при движении слайдера
- Кнопка поднимается при наведении

## Технические детали

### Вычисление цвета:
```javascript
const getGradientColor = (value) => {
    const r = Math.round(139 * value);
    const g = Math.round(92 * value);
    const b = Math.round(246 * value);
    return `rgb(${r}, ${g}, ${b})`;
};
```

### Градиент полосы:
```javascript
background: `linear-gradient(90deg, 
    #000000 0%, 
    ${color} ${confidence * 100}%, 
    rgba(255,255,255,0.1) ${confidence * 100}%
)`
```

Это создает эффект заполнения полосы от черного до фиолетового.

## Сохранение

В объекте команды сохраняются:
- `name` - название команды
- `role` - выбранная роль
- `confidence` - уровень уверенности (0-1)
- `backgroundColor` - вычисленный цвет в формате `rgb(r, g, b)`

Эти данные используются для отображения градиента на странице "Моя команда".
