# Контейнер iPhone 13 Mini для страницы анкеты

## Описание

Создан HTML-контейнер с CSS для имитации дисплея iPhone 13 Mini на странице анкеты.

## Структура

```html
<div class="iphone-container-q">      <!-- Корпус телефона -->
    <div class="phone-screen-q">       <!-- Экран телефона -->
        <div class="form-container-q">  <!-- Контент с эффектами -->
            <!-- Aurora эллипсы -->
            <div class="glass-card-q">  <!-- Стеклянная карточка -->
                <!-- Форма анкеты -->
            </div>
        </div>
    </div>
</div>
```

## Характеристики

### 1. Корпус телефона (.iphone-container-q)
- **Размеры**: 375px × 812px (стандартные размеры iPhone 13 Mini)
- **Фон**: Темный (#1a1a1a) - имитация рамки телефона
- **Закругление**: 40px - закругленные углы корпуса
- **Padding**: 12px - отступ для создания рамки
- **Тень**: Глубокая тень для объемности

### 2. Экран телефона (.phone-screen-q)
- **Размеры**: 100% от контейнера (351px × 788px после padding)
- **Фон**: Черный (#000000)
- **Закругление**: 32px - закругленные углы экрана
- **Прокрутка**: `overflow-y: auto` - вертикальная прокрутка контента
- **Скрытие скроллбара**: Полоса прокрутки скрыта, но функциональность сохранена

### 3. Контейнер контента (.form-container-q)
- **Размеры**: 100% от экрана
- **Позиция**: Относительная
- **Overflow**: Visible - для Aurora эффектов

## CSS Код

```css
/* Контейнер iPhone 13 Mini */
.iphone-container-q {
    width: 375px;
    height: 812px;
    background: #1a1a1a;
    border-radius: 40px;
    padding: 12px;
    box-shadow: 0 20px 60px rgba(0, 0, 0, 0.8);
    position: relative;
}

/* Экран телефона */
.phone-screen-q {
    width: 100%;
    height: 100%;
    background: #000000;
    border-radius: 32px;
    overflow-y: auto;
    overflow-x: hidden;
    position: relative;
    scrollbar-width: none;
    -ms-overflow-style: none;
}

.phone-screen-q::-webkit-scrollbar {
    display: none;
}
```

## Функциональность

### Прокрутка контента
- Контент внутри `.phone-screen-q` прокручивается вертикально
- Полоса прокрутки скрыта для чистого вида
- Прокрутка работает мышью, тачпадом и на мобильных устройствах

### Центрирование
- Контейнер iPhone центрирован на странице
- Использует flexbox для выравнивания по центру

### Адаптивность
- Фиксированные размеры 375×812px
- Padding страницы 20px для отступов от краев
- Min-height: 100vh для полной высоты экрана

## Визуальные эффекты

### Корпус телефона
- Темная рамка имитирует корпус iPhone
- Глубокая тень создает эффект объема
- Закругленные углы соответствуют дизайну iPhone

### Экран
- Черный фон как у настоящего iPhone
- Внутренние закругления меньше внешних
- Aurora эффекты внутри экрана

## Размеры

| Элемент | Ширина | Высота |

Контейнер автоматически применяется на странице анкеты (QuestionnairePage). Весь контент формы отображается внутри имитации iPhone 13 Mini с возможностью прокрутки.
