# Подключение к бэкенду

## Что сделано:

### 1. AuthForm.jsx (Страница входа)
- Подключен эндпоинт `POST /api/auth/dev-login`
- Сохраняет токен в localStorage
- Простая обработка ошибок через alert

### 2. Questionnaire.jsx (Анкета)
- Подключен эндпоинт `PUT /api/participants/me`
- Отправляет данные профиля на бэкенд
- Конвертирует навыки в массив

### 3. Feed.jsx (Свайп)
- Загружает участников: `GET /api/participants?hackathonId=h1`
- Загружает команды: `GET /api/teams?hackathonId=h1`
- Отправляет свайпы: `POST /api/swipe`
- Показывает алерт при мэтче

### 4. MatchList.jsx (Мэтчи)
- Загружает мэтчи: `GET /api/matches?participantId={id}`
- Показывает список команд с мэтчами

### 5. TeamCreation.jsx (Создание команды)
- Создает команду: `POST /api/teams`
- Конвертирует роли в массив

## Как тестировать:

### Шаг 1: Запустить бэкенд
```bash
cd Courses_hack_repository/itam_hack
python main.py
```
Бэкенд запустится на http://localhost:8000

### Шаг 2: Запустить фронтенд
```bash
cd frontend
npm run dev
```
Фронтенд запустится на http://localhost:5174

### Шаг 3: Войти в систему
1. Открыть http://localhost:5174
2. Ввести код: `u1` (или `u2`, `u3`, `u4`)
3. Нажать "Войти"

### Шаг 4: Заполнить анкету
1. Заполнить все поля
2. Данные сохранятся на бэкенде

### Шаг 5: Выбрать хакатон
1. Выбрать любой хакатон
2. Нажать "Хочу участвовать"

### Шаг 6: Свайпать
1. Свайпать карточки
2. При лайке данные отправляются на бэкенд
3. Если есть взаимный лайк - появится алерт "Это мэтч!"

### Шаг 7: Посмотреть мэтчи
1. Нажать "Мэтчи" в нижнем меню
2. Увидеть список команд с мэтчами
