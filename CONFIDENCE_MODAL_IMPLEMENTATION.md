# Реализация модального окна оценки уверенности

## Описание

Модальное окно для самооценки уверенности в выбранной роли, которое появляется сразу после выбора роли в анкете.

## Триггер

Окно появляется автоматически, когда пользователь выбирает роль из выпадающего списка в форме анкеты.

## Дизайн и Компоненты

### 1. Модальное окно
- **Тип**: Всплывающее окно с закругленными краями (24px)
- **Позиция**: Центрировано на экране
- **Фон**: Темный (почти черный) `rgba(20, 20, 20, 0.95)`
- **Оверлей**: Полупрозрачный черный с размытием
- **Анимация**: Плавное появление снизу вверх

### 2. Заголовок
- **Текст**: "Насколько ты уверен в своих способностях?"
- **Стиль**: Крупный (28px), жирный (700), белый цвет
- **Выравнивание**: Слева

### 3. Выбранная роль
- **Отображение**: Фиолетовая кнопка с градиентом
- **Элементы**: 
  - Галочка (✓) слева
  - Название роли (например, "Product-менеджер")
- **Цвета**: Градиент от `#C026D3` до `#A855F7`
- **Тень**: Фиолетовое свечение

### 4. Шкала уверенности

**Структура:**
- Черный квадрат слева (50x50px)
- Градиентная полоса посередине
- Проценты справа

**Градиентная полоса:**
- **Фон трека**: От черного к полупрозрачному фиолетовому
- **Заполнение**: Градиент от черного через фиолетовый к светло-фиолетовому
- **Ползунок**: Белый круг с фиолетовым свечением
- **Высота**: 12px (трек), 30px (ползунок)

**Проценты:**
- **Цвет**: Фиолетовый `#C026D3`
- **Размер**: 24px, жирный
- **Эффект**: Текстовое свечение

### 5. Кнопка "Подтвердить"
- **Размер**: Полная ширина, 18px padding
- **Цвет**: Яркий фиолетовый градиент
- **Форма**: Закругленная (50px radius)
- **Текст**: "Подтвердить" (22px, жирный)
- **Эффект**: Поднимается при наведении

## Функциональность

### Взаимодействие:
1. Пользователь выбирает роль из выпадающего списка
2. Автоматически появляется модальное окно
3. Пользователь перемещает ползунок по градиентной шкале
4. Проценты обновляются в реальном времени
5. Нажимает "Подтвердить"
6. Роль сохраняется, окно закрывается

### Закрытие окна:
- Клик на "Подтвердить" - сохраняет роль и закрывает
- Клик вне окна - закрывает без сохранения роли

## Технические детали

### Состояния компонента:
```javascript
const [showConfidenceModal, setShowConfidenceModal] = useState(false);
const [confidence, setConfidence] = useState(0.8); // 80% по умолчанию
const [tempRole, setTempRole] = useState(''); // Временное хранение выбранной роли
```

### Обработчики:
```javascript
// При выборе роли
const handleRoleChange = (e) => {
    const selectedRole = e.target.value;
    if (selectedRole && selectedRole !== formData.role) {
        setTempRole(selectedRole);
        setShowConfidenceModal(true);
    }
};

// При подтверждении
const handleConfirmConfidence = () => {
    setFormData({ ...formData, role: tempRole });
    setShowConfidenceModal(false);
};
```

### Градиент заполнения:
```css
background: linear-gradient(90deg, 
    #000000 0%,      /* Черный слева */
    #C026D3 50%,     /* Фиолетовый в центре */
    #A855F7 100%     /* Светло-фиолетовый справа */
);
```

## Анимации

1. **Появление окна**: 
   - Плавное увеличение и движение снизу вверх
   - Длительность: 0.4s

2. **Заполнение шкалы**: 
   - Плавное изменение ширины
   - Длительность: 0.1s

3. **Кнопка**: 
   - Поднимается на 2px при наведении
   - Усиление тени

4. **Свечение**: 
   - Фиолетовое свечение на кнопке роли
   - Свечение на заполнении шкалы
   - Текстовое свечение на процентах
