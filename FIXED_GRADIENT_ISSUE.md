# Исправление: Градиентная шкала

## Что было исправлено:

### 1. Убран слайдер из анкеты
- **Проблема**: Слайдер уверенности отображался в форме анкеты (QuestionnairePage)
- **Решение**: Полностью удален блок со слайдером из анкеты
- **Результат**: В анкете теперь только выбор роли из выпадающего списка, без слайдера

### 2. Градиент отображается правильно
- **Где**: Градиент от черного к фиолетовому применяется на странице "Моя команда" (MyTeamPage)
- **Как работает**: 
  1. При создании команды выбираешь роль
  2. В модальном окне двигаешь слайдер (выбираешь уровень уверенности)
  3. Цвет вычисляется на основе уровня: `rgb(139 * confidence, 92 * confidence, 246 * confidence)`
  4. Этот цвет сохраняется в команде
  5. На странице "Моя команда" фон использует градиент: `linear-gradient(180deg, #000000 0%, [выбранный цвет] 100%)`

### 3. Модальное окно - отдельное
- **Glassmorphism окно**: Появляется поверх всего как отдельное модальное окно
- **Не в анкете**: Модальное окно не является частью формы анкеты
- **Триггер**: Появляется только при создании команды, после выбора роли

## Как проверить:

1. **Анкета (QuestionnairePage)**:
   - Зайди в анкету
   - Проверь, что там НЕТ слайдера уверенности
   - Должен быть только выпадающий список с ролями

2. **Создание команды (CreateTeamPage)**:
   - Создай команду
   - Выбери роль из списка
   - Появится glassmorphism модальное окно
   - Подвигай слайдер - увидишь изменение процентов
   - Нажми "Подтвердить"

3. **Моя команда (MyTeamPage)**:
   - Перейди на страницу "Моя команда"
   - Фон должен быть градиентом от черного к фиолетовому
   - Интенсивность фиолетового зависит от выбранного уровня уверенности

## Примеры градиентов:

- **Уверенность 0.2 (20%)**: Очень темный, почти черный
- **Уверенность 0.5 (50%)**: Темно-фиолетовый
- **Уверенность 0.8 (80%)**: Средне-фиолетовый (по умолчанию)
- **Уверенность 1.0 (100%)**: Яркий фиолетовый `rgb(139, 92, 246)`
