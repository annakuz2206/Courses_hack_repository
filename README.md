# HackTeam - Платформа поиска команд для хакатонов

Простой и красивый фронтенд для поиска команд на хакатоны.
Оптимизирован под **iPhone 13 mini (375x812px)**.

##  Запуск

1. Установи Node.js
- Скачай с nodejs.org версию 18 или выше

2. Скачай проект
```bash
   git clone https://github.com/annakuz2206/Courses_hack_repository.git
```
3. Установи зависимости, установи и запускай

```bash
cd Courses_hack_repository
npm install
npm run dev
```

Откроется на `http://localhost:5174`

## User Flow

```
1. Вход → Введи имя
2. Анкета → Заполни профиль
3. Хакатоны → Выбери хакатон (клик → модалка с описанием)
4. Статус → Есть команда? Да/Нет
   ├─ Да → Расскажи о команде → Свайп
   └─ Нет → Искать или Создать → Свайп
5. Свайп → Лайкай команды/участников
6. Мэтчи → Общайся в чате
```

## Страницы

1. **Вход** (`/`) - Простая форма входа
2. **Анкета** (`/questionnaire`) - Заполнение профиля
3. **Хакатоны** (`/hackathon-selection`) - Список с модалками
4. **Статус команды** (`/team-status`) - Есть команда?  НОВОЕ
5. **Создание команды** (`/team-creation`) - Форма для капитанов
6. **Свайп** (`/feed`) - Главный экран
7. **Детали** (`/card/:id`) - Подробная информация
8. **Мэтчи** (`/matches`) - Список совпадений
9. **Чат** (`/chat/:id`) - Общение
10. **Настройки** (`/settings`) - Редактирование
11. **Админка** (`/admin`) - Статистика

## Технологии

- React 19
- React Router 7
- Vite 7
- Чистый CSS (без библиотек)

## 🍽️ Технологический стек

- Backend: FastAPI (Python 3.11+)
- Аутентификация: JWT + Telegram OAuth
- Документация: Swagger UI / ReDoc
- Хранилище: In-memory (для MVP), готово к миграции на PostgreSQL
- Контейнеризация: Docker (опционально)
