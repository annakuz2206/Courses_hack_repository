# Инструкция: Glassmorphism модальное окно уверенности

## Дизайн как на картинке

Модальное окно теперь имеет эффект glassmorphism (стеклянное размытие) и выглядит точно как на предоставленной картинке.

## Элементы дизайна:

### 1. Фон
- Полупрозрачный черный с размытием (backdrop-filter: blur)
- Создает эффект глубины

### 2. Glassmorphism окно
- Полупрозрачный белый фон: `rgba(255, 255, 255, 0.1)`
- Размытие: `backdrop-filter: blur(20px)`
- Тонкая белая граница: `rgba(255, 255, 255, 0.2)`
- Закругленные углы: 24px

### 3. Заголовок
- Белый текст: "Насколько ты уверен в своих способностях?"
- Размер: 24px
- Выравнивание: слева

### 4. Кнопка с ролью
- Фиолетовый градиент: `#C026D3` → `#A855F7`
- Галочка слева + название роли
- Закругленные углы: 16px
- Тень с фиолетовым свечением

### 5. Слайдер (главный элемент)
Состоит из трех частей:

**Левый квадрат:**
- Размер: 50x50px
- Фиолетовый градиент
- Закругленные углы: 12px
- Тень с свечением

**Трек слайдера:**
- Серый фон: `rgba(255, 255, 255, 0.2)`
- Заполнение: фиолетовый градиент
- Белый кружок (ползунок)
- Плавная анимация заполнения

**Проценты справа:**
- Фиолетовый цвет: `#C026D3`
- Размер: 24px, жирный
- Свечение текста
- Показывает текущее значение (например, 41%)

### 6. Кнопка "Подтвердить"
- Яркий фиолетовый градиент
- Полная ширина
- Размер: 22px, жирный
- Закругленные углы: 16px
- Тень с фиолетовым свечением
- Анимация при наведении

## Цветовая палитра:

- **Основной фиолетовый**: `#C026D3` (фуксия)
- **Светлый фиолетовый**: `#A855F7`
- **Белый текст**: `#FFFFFF`
- **Полупрозрачный фон**: `rgba(255, 255, 255, 0.1)`

## Анимации:

1. **Появление окна**: Плавное увеличение снизу вверх
2. **Заполнение слайдера**: Плавное изменение ширины
3. **Кнопка**: Поднимается при наведении
4. **Свечение**: Тени с фиолетовым свечением

## Как использовать:

1. Создайте команду и введите название
2. Выберите роль из списка
3. Появится glassmorphism окно
4. Двигайте белый кружок по слайдеру
5. Наблюдайте изменение процентов справа
6. Нажмите "Подтвердить"

## Технические детали:

### Glassmorphism эффект:
```css
background: rgba(255, 255, 255, 0.1);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.2);
```

### Слайдер с белым кружком:
```css
/* Невидимый input range поверх трека */
opacity: 1;
/* Белый кружок как thumb */
background: white;
border-radius: 50%;
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.3);
```

### Фиолетовое свечение:
```css
box-shadow: 0 4px 16px rgba(192, 38, 211, 0.4);
text-shadow: 0 0 10px rgba(192, 38, 211, 0.5);
```

## Результат:

Модальное окно выглядит как стеклянная панель, парящая над размытым фоном, с яркими фиолетовыми акцентами и плавными анимациями. Точно как на картинке!
